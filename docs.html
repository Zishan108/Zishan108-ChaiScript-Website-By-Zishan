List Items

List items are ordered, changeable, and allow duplicate values.

List items are indexed, the first item has index [0], the second item has index [1] etc.



Ordered

When we say that lists are ordered, it means that the items have a defined order, and that order will not change.

If you add new items to a list, the new items will be placed at the end of the list.

Note: There are some list methods that will change the order, but in general: the order of the items will not change.


Changeable

The list is changeable, meaning that we can change, add, and remove items in a list after it has been created.

Allow Duplicates

Since lists are indexed, lists can have items with the same value:

cup my_list = ["apple", "banana", "cherry", "apple"];

#output: ["apple", "banana", "cherry", "apple"]

ChaiScript Conditions and If statements

ChaiScript supports the usual logical conditions from mathematics:

  • Equals: a == b

  • Not Equals: a != b

  • Less than: a < b

  • Less than or equal to: a <= b

  • Greater than: a > b

  • Greater than or equal to: a >= b

These conditions can be used in several ways, most commonly in "if statements" and loops.

An "if statement" is written by using the hot keyword.

Example:

cup a = 12;
cup b = 1212;
hot b > a then_chai
   serveChai("b is greater than a")
sip

In this example we use two variables, a and b, which are used as part of the if statement to test whether b is greater than a. As a is 12, and b is 1212, we know that 1212 is greater than 12, and so we print to screen that "b is greater than a".

Note:
1. The then_chai keyword is used to define a code block
Here, The code block is if statement (hot)

2. The sip keyword is used for defining the end of a particular code block
Here, we are writing sip after writing the whole logic of our if condition which means that the scope of the if (hot) ends here
You won't write the sip keyword while writing the code on terminal directly as we write code one line at a time hence there is no concept of block termination exist


else if

The mild keyword is ChaiScript's way of saying "if the previous conditions were not true, then try this condition".

Example:

cup a = 12;
cup b = 12;

hot b > a then_chai
   serveChai("b is greater than a");
mild b == a then_chai
   serveChai("b is equal to a");
sip

In this example a is equal to b, so the first condition is not true, but the elif condition is true, so we print to screen that "a and b are equal".

Note: Notice how the sip keyword is only written once for both hot and mild. It's because ChaiScript Considers every conditional statement as One Scope revolving around a single condition.


else

The chilled keyword catches anything which isn't caught by the preceding conditions.

Example:

cup a = 12;
cup b = 12;
hot a > b then_chai
  serveChai("a is greater than b");

mild a < b then_chai
  serveChai("a is smaller than b");

chilled
  serveChai("a is Equals to b");
sip;

And

The and keyword is a logical operator, and is used to combine conditional statements:

It evaluates true if both the conditions are true else it evaluates it to false

Example:

cup a = 19;

hot a > 18 and a < 100 then_chai
  serveChai("Valid age! Both Conditions are true");
sip;

Or

The or keyword is a logical operator, and is used to combine conditional statements:

It evaluates true even if one of the condition is true. It only evaluates to false if both the condition is false.

Example:

cup a = 19;

hot a > 18 or a <= 19 then_chai
  serveChai("Valid age!");
sip;

Not

The not keyword is a logical operator, and is used to reverse the result of the conditional statement:

Example:

cup a = 19;

hot not a < 18 then_chai
  serveChai("Valid age!");
sip;

Functions

Functions in ChaiScript are defined using the brew keyword. They allow you to encapsulate reusable blocks of code.

Function Declaration

Basic function syntax with a code block:

brew greet(name)
  serveChai("Hello " + name)
sip



One-line Function

For simple functions, you can use arrow syntax:

brew square(x) -> x * x



Calling Functions

Functions are called using parentheses:

cup result = square(5)
serveChai(result) # Output: 25



Parameters

Functions can accept multiple parameters:

brew add(a, b)
  serveChai(a + b)
sip



Note: The sip keyword is only required for multi-line function blocks. One-line functions using arrow syntax don't need sip.

Built-in Functions

This section covers the built-in functions available in ChaiScript, categorized by their functionality. Each function includes a description and example usage.

Utility Functions

These functions handle basic input/output, type conversion, and program flow control.

1. serveChai(arg) - Prints the argument to the screen.

serveChai("Hello, Chai!") # Output: Hello, Chai!


2. serve_ret(arg) - Prints and returns the argument.

cup x = serve_ret("Hi") # Prints: Hi, Assigns "Hi" to x


3. orderChai(arg) - Prints the argument as a prompt and takes user input (returns string).

cup name = orderChai("Enter name: ") # Prompts user, assigns input to name


4. orderChai_int(string) - Prompts with the string and takes numeric input (returns integer).

cup age = orderChai_int("Enter age: ") # Prompts user, assigns integer to age


5. clean() / cls() - Clears the ChaiScript terminal.

clean(); # Clears the terminal
cls(); #also clears the terminal


6. type_chai(arg) - Returns the type of the argument.

serveChai(type_chai(42)) # Output: Number


7. current_date_time() - Returns the current date and time.

serveChai(current_date_time()) # Output: e.g., "2025-02-21 10:00:00"


8. list_len(list) - Returns the length of a list.

cup lst = [1, 2, 3]; serveChai(list_len(lst)) # Output: 3


9. str_len(string) - Returns the length of a string.

serveChai(str_len("Chai")) # Output: 4


10. len(arg) - Returns the length of any datatype.

serveChai(len("Tea")) # Output: 3


11. int(arg) - Converts the argument to an integer.

cup num = int("123"); serveChai(num) # Output: 123

12. str(arg) - Converts the argument to a string.

serveChai(str(42)) # Output: "42"

13. float(arg) - Converts the argument to a float.

cup f = float("3.14"); serveChai(f) # Output: 3.14

14. time_freeze(int) - Pauses execution for the specified seconds.

serveChai("Waiting..."); time_freeze(2); serveChai("Done!") # Waits 2 seconds

Boolean Functions

These functions return true or false based on type or emptiness checks.

1. is_chai_num(arg) - Returns true if the argument is a number.

serveChai(is_chai_num(42)) # Output: true

2. is_chai_str(arg) - Returns true if the argument is a string.

serveChai(is_chai_str("Chai")) # Output: true

3. is_chai_list(arg) - Returns true if the argument is a list.

serveChai(is_chai_list([1, 2])) # Output: true

4. is_brew(arg) - Returns true if the argument is a function.

brew my_func() -> 42; serveChai(is_brew(my_func)) # Output: true

5. is_empty(arg) - Returns true if the argument is empty.

serveChai(is_empty("")) # Output: true

String Functions

These functions manipulate or inspect strings.

1. to_lower(string) - Converts string to lowercase.

serveChai(to_lower("CHAI")) # Output: "chai"

2. to_upper(string) - Converts string to uppercase.

serveChai(to_upper("chai")) # Output: "CHAI"

3. char_at(string, index) - Returns character at index.

serveChai(char_at("Chai", 1)) # Output: "h"

4. index_of(string1, string2) - Returns index of string2 in string1.

serveChai(index_of("ChaiScript", "Script")) # Output: 4

5. slice(string, start, end, step) - Returns a substring.Takes 1st arg as the primary string, 2nd as start index, third as end index and fourth as hops or steps over the indexes.

serveChai(slice("ChaiScript", 0, 4, 1)) # Output: "Chai"

6. starts_with(string1, string2) - Checks if string1 starts with string2.

serveChai(starts_with("Chai", "Ch")) # Output: true

7. ends_with(string1, string2) - Checks if string1 ends with string2.

serveChai(ends_with("Chai", "ai")) # Output: true

8. trim(string) - Removes trailing spaces.

serveChai(trim(" Chai ")) # Output: "Chai"

9. replace(string, old, new) - Replaces All occurrences of old substring with new substring.

serveChai(replace("Tea Tea", "Tea", "Chai")) # Output: "Chai Chai"

10. split_to_list(string, delimiter) - Splits string into a list with the specified Delimitter.

serveChai(split_to_list("a,b,c", ",")) # Output: ["a", "b", "c"]

11. capitalize(string) - Capitalizes the first letter.

serveChai(capitalize("chai")) # Output: "Chai"

12. title(string) - Converts to title case.

serveChai(title("chai script")) # Output: "Chai Script"

List Functions

These functions operate on lists.

1. append(list, arg) - Adds an element to the list.

cup lst = [1, 2]; append(lst, 3); serveChai(lst) # Output: [1, 2, 3]

2. pop(list, index) - Removes and returns the element at index.

3. extend(list1, list2) - Adds list1 elements to list2.

6. range(start, end, step) - Creates a list of numbers.

7. element_at(list, index) - Returns element at index.

8. join_to_str(list, delimiter) - Joins list into a string.

cup lst = ["a", "b", "c"]; serveChai(join_to_str(lst, "-")) # Output: "a-b-c"

For Loop

For loop is used when you want to iterate a particular code for a specific number of times.

The keyword for the for loop in ChaiScript is for_chai.

Syntax:

for_chai i = start to_chai end then_chai
  # code block
sip

Example:

This example demonstrates a for loop that iterates from 0 to 6 and prints each value:

for_chai i = 0 to_chai 7 then_chai
  serveChai(i)
sip

Output:

0
1
2
3
4
5
6

The to_chai keyword is used to declare the last iteration number. It is exclusive, meaning the loop runs up to but does not include this number. For example, if to_chai is set to 7, the loop will run from 0 to 6.

Using STEP Keyword:

The STEP keyword is used to specify the increment or decrement value for the loop variable. It allows you to control the step size of the loop.

Example:

This example demonstrates a for loop that iterates from 5 to 1 in steps of -1 and prints each value:

for_chai i = 5 to_chai 0 STEP -1 then_chai
  serveChai(i)
sip

Output:

5
4
3
2
1

While Loop

The while loop is used to execute a block of code as long as a specified condition is true.

The keyword for the while loop in ChaiScript is while_chai.

Syntax:

while_chai condition then_chai
  # code block
sip

Example:

This example demonstrates a while loop that iterates as long as the variable a is less than or equal to 5 and prints each value:

cup a = 1;

while_chai a <= 5 then_chai
  serveChai(a);
  cup a = a + 1;
sip

Output:

1
2
3
4
5
